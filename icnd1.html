  • Physical Layer

    Specifies hardware and signalling characteristics of network media such as materials, electrical or RF characteristics, interfaces, etc.

  • Data Link Layer

    Performs framing of network data to be transmitted, identifying the protocol type of the packet and providing error control.

  • Internet Layer

    Maintains knowledge of network topography and adds source and destination network information (datagram header) to packet for delivery over remote links

  • Transport Layer

    Prepends segment header data to packet to multiplex sessions or virtual circuits i.e. TCP or UDP port numbers

  • Application Layer

    Provides services to user and system applications necessary to access or troubleshoot the network

1.2 Compare and contrast TCP and UDP protocols

https://en.wikibooks.org/wiki/Communication_Networks/TCP_and_UDP_Protocols

  • TCP

    • Connection-oriented
    • Complex
    • Reliable
    • Bidirectional (Full-Duplex)
    • Retransmission
    • Sequencing
    • Rate adaptation

  • UDP

    • Connectionless
    • Unreliable
    • Fast
    • Simple
    • No retransmissions or flow control
    • Data sent as discrete messagess with no sequencing information provided by protocol

1.3 Describe the impact of infrastructure components in an enterprise network

  • 1.3.a Firewalls

    • Filter traffic to and from unknown networks based on rules such as ip addresses and port numbers
    • May be stateful or stateless, inspecting, and operate at different layers
    • Historically often implemented at routers/network boundaries

  • 1.3.b Access points

    • Wireless host network access
    • May be autonomous or connected to a controller

  • 1.3.c Wireless controllers

    • Coordinate access points for extended wireless networking

1.5 Compare and contrast network topologies

  • 1.5.a Star

    • Devices connect to central hub
    • Failure domain limited to each link

  • 1.5.b Mesh

    • Each device has a connection to every other device
    • Full redundancy
    • Expense and complexity rise geometrically

  • 1.5.c Hybrid

    • Combination of approaches
    • Maximize strengths of each, minimize drawbacks of each
